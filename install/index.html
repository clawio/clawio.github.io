Install

Learn how to install ClawIO.

ClawIO Daemon

The Daemon is the core component of the ClawIO Project. It provides a HTTP/HTTPS server with a REST API.

The fastest way to download the clawiod binary is through GitHub Release Pages.

The releases are provided in the following formats:

  • DEB package
  • Binary for Mac OSX 64 bits
  • Binary for Linux 64 bits
  • Source code

In order to compile the source code you need Go installed.
